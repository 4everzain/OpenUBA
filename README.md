## Goals
To Build a lightweight, SIEM Agnostic, UEBA Framework focused on providing:
- Model Management
- Community-driven Model Library
- Model Versioning
- Dashboard
- Rule Storage/Management
- Case Management
- Peer-oriented/community intel
- Lightweight, SIEM-agnostic infrastructure
- Flexible/open dataset support
- Alerting/Ticketing system

## Components
- Client Dashboard
  - React
  - Bootstrap
  - Node JS
  - Express JS
  - D3.js
- Model Server (Remote or Local)
- User Interface
- API Server
  - Flask
- Visualization
  - Data Shader
  - Kibana
- Modeling
  - Tensorflow
  - Sci-kit Learn
  - Keras
  - GP Learn
  - DEAP
  - Network X
  - MLLib
- Compute Engine
  - Spark
  - Elastic Search
- Log Transport
  - Logstash
-Log Collection
  - Beats
- Supported Data Formats (for now)
  - CSV
  - Parquet
  - Flat File
